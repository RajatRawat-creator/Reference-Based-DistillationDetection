Question:
The sequences of positive integers $1,a_2, a_3,...$ and $1,b_2, b_3,...$ are an increasing arithmetic sequence and an increasing geometric sequence, respectively. Let $c_n=a_n+b_n$ . There is an integer $k$ such that $c_{k-1}=100$ and $c_{k+1}=1000$ . Find $c_k$ .

Answer:
To solve this problem, we need to understand the properties of the sequences involved.

Given:
1. \( 1, a_2, a_3, \ldots \) is an increasing arithmetic sequence.
2. \( 1, b_2, b_3, \ldots \) is an increasing geometric sequence.
3. \( c_n = a_n + b_n \).
4. There exists an integer \( k \) such that \( c_{k-1} = 100 \) and \( c_{k+1} = 1000 \).

We need to find \( c_k \).

### Analysis:
1. Given \( c_n = a_n + b_n \), 
2. For the arithmetic sequence, \(a_n = a_1 + (n-1)r\) where \(a_1 = 1\), \(r\) is the common difference, and \(a_n > 1\) for all \(n\).
3. For the geometric sequence, \(b_n = b_1 \cdot r^{n-1}\) where \(b_1 = 1\) and \(r\) is the common ratio, and \(b_n > 1\) for all \(n\).
4. Given \(c_{k-1} = 100\), \(c_{k+1} = 1000\).
5. Hence, \(c_n = 1000\) when \(n = k - 1 + 1 = k\), since \(c_{k-1} = 100\), \(c_{k-1} + 2r = 100 + 2r\), and since \(c_{k+1} = 1000\), \(c_{k+1} + 2r = 1000 + 2r\). Since \(c_{k-1} + 2r = 100 + 2r\), and \(c_{k+1} + 2r = 1000 + 2r\), we have \(100 + 2r = 100 + 2r\), and \(1000 + 2r = 1000 + 2r\), thus \(c_{k-1} + 2r = 100\), and \(c_{k+1} + 2r = 1000\), since \(c_n = a_n + b_n\), we have \(c_{k-1} + c_{k+1} = a_{k-1} + b_{k-1} + a_{k+1} + b_{k+1}\).

Given \(a_{k-1} + a_{k+1} = 2a_1 = 2\), and \(b_{k-1} + b_{k+1} = 2b_1 = 2\), thus \(c_{k-1} + c_{k+1} = 0\), thus \(c_k = 0\).

Thus, the final answer is \(\boxed{0}\).
